# Cole Dustin's Portfolio (GitHub Pages)

This is a static site (HTML/CSS/JS) designed to be uploaded directly to GitHub Pages.

## Quick start (User/Org site)
1. Create a repo named: `YOUR_USERNAME.github.io`
2. Upload everything in this folder to the repo root
3. In GitHub: Settings → Pages → Source: `Deploy from a branch` → Branch: `main` → Folder: `/ (root)`
4. Your site will be live at: `https://YOUR_USERNAME.github.io`

## Quick start (Project site)
1. Create any repo, upload this folder contents
2. Settings → Pages → Branch: `main` → Folder: `/ (root)`
3. Your site will be at: `https://YOUR_USERNAME.github.io/REPO_NAME/`

## Where to put things
- Images: `assets/img/`
- Project reports / PDFs: `assets/docs/projects/`
- Reflection PDFs: `assets/docs/reflections/`
- Resume PDF: `assets/docs/resume/resume.pdf`  (replace placeholder)

## Editing
- Main page: `index.html`
- Project pages: `projects/*/index.html`
- Resume: `resume/index.html`
- Reflections: `reflections/index.html`
- Contact: `contact/index.html`
- Styles: `assets/css/styles.css`
